# 📊 Data Analyst Assignment

## 🔹 Overview

Basic data analysis project using SQL and Excel to create tables, run queries, and extract insights.

## 🛠️ Tools

SQL | Excel | Power BI | Python

## 📂 Features

* Table creation (Users, Items, Bookings)
* SQL queries (SELECT, WHERE, GROUP BY)
* Data analysis using aggregate functions

## 📌 Sample Query

```sql
SELECT name, COUNT(*)
FROM details_data
GROUP BY name;
```

Improved SQL skills, data analysis, and problem-solving.
